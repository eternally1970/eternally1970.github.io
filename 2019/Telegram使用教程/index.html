设置代理

PC端

首先在登陆界面点击右上角的Setting

接着在弹出的界面里面依次选择Connection Type - Use custom proxy - MTPROTO对照我给你们的代理信息填进去

如果连接成功下面会提示Online，若长时间连接不上请尝试换用其他节点

安卓端

爆肝中，整体上与PC大同小异，设置的入口是左上角

登录

回到最开始的页面，点击小箭头，在弹出的对话框中输入China 选择+86
然后输入自己的手机号，按照提示完成登陆（短信验证码来自国外，接受需要较长的时间）

汉化与使用

登录Telegram后，在左上角的搜索框输入@zh_CN进入第一个群组，直接点击那串链接就可以安装中文语言包，完成汉化

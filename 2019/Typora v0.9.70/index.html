Markdown 是一种由美利坚用户界面设计师 John Gruber 开发的轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的 XHTML 或者 HTML 文档。这种语言吸收了很多在电子邮件中已有的纯文本标记的特性。由于 Markdown 的轻量化与易读易写特性，并且对于图片，图表、数学式都有支持，当前许多网站都广泛使用 Markdown 来撰写帮助文档或是用于论坛上发表文章。

Typora 是一款由中国上海爱好设计、编码与游戏的软件工程师 Abner Lee 精心设计与开发出品的完全免费的轻量级 Markdown 编辑阅读器，而它与其他的 Markdown 编辑器不同的是，Typora 没有采用源代码和预览双栏显示的方式，而是采用了所见即所得的编辑方式，进而实现即时预览的功能，但也可切换至源代码编辑模式。Markdown编辑阅读器

功能

  • 所见即所得 – Typora 设计理念就是极致简洁，把其他编辑器的预览窗口都省了，同时还支持实时预览；
  • 快捷键 – 便捷的表格、代码、公式编辑；
  • 目录 – 在文章中使用 [TOC] 自动根据层级生成目录；
  • 图片拖拽 – 拖入图片即可插入图片，但是这个功能默认是关闭的需要在设置中手动开启；
  • 自定义主题 – 自带五套主题方案，支持通过自定义 CSS 方式进行个性化定制；
